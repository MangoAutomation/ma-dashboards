This is a complete Material Design admin template, similar to these documentation pages, with a menu and toolbar which you can customize.

Copy the entire ${MA_HOME}/web/modules/ui/web/adminTemplate folder into the ${MA_HOME}/overrides/web/modules/ui/web directory, give it a new name then customize the menu and other behavior by editing app.js (shown below).

You will need change two references to adminTemplate in index.html to the name of the folder which you created in overrides.

You can add your own views, controllers, directives and filters. You may also want to define some of the constants shown on the extend app page
