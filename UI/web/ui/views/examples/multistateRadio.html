This example showcases using angular material radio buttons to set the value of multistate data points. Set the ng-model to the points valueFn property and set ng-model-options="{getterSetter:true}".

The first example shows how to loop through the text renderer map of the multistate point to get the various options set on the data point properties. The second example shows how to use fixed values of 0-5.

See API Documentation for the valueFn and rendererMap methods on the Point object.
