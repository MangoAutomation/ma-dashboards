Google Maps

Google maps can be added to your custom pages using the <ma-map> component. Under the hood <ma-map> uses the ng-map library which allows you to nest <marker> and <info-window> components inside <ma-map>. The example shown here demonstrates how you can use markers to toggle Info Windows that contain other Mango Components, change marker icons, set initial marker and map location (lat/long), make markers draggable, and use a marker to set output-data that can update other parts of your page.

* Note: In order to use the <ma-map> component you must set up your own Google Maps API Key. You can get your key from Google then enter it into the Dashboard Settings page.

See API Documentation for <ma-map>.

  • First Marker: Clicking toggles an info window which sets a binary point's value to "{{myPoint1.value}}" and uses its value to switch its icon.
  • Green Marker: Clicking toggles an info window which sets a numeric point's value to {{myPoint2.value}}.
  • Red Marker: Clicking toggles an info window which displays a small chart of latest 50 values.
  • Yellow Marker: Sets output-data variable to "{{myOutputData}}".
