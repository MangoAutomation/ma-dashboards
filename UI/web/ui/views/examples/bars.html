This examples shows two different ways to create a bar to display the value of a data points. You can use an Angular Material Progress Linear directive and pass the value to ng-value, or create your own using CSS and the ng-style directive.

You can use the .bar-display CSS classes used in this example as a starting point, please see ngMango.css.

See API Documentation for <ma-get-point-value>.
