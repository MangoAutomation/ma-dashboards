Binary data points can be set using various methods, some of which are shown below. The toggleValue() method can be used on a point, which works great via a ng-click on a <ma-switch-img> directive.

HTML checkbox and angular material switch components can be used as well. Set the ng-model to the points valueFn property and set ng-model-options="{getterSetter:true}".

See API Documentation for <ma-switch-img> and valueFn / toggleValue() methods on the Point object.
