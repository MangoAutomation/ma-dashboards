The Watch List page allows you to compare many data points at the same time in a table view. Powerful query and parameter based watchlists can be built using the Watch List Builder, and later viewed on the this page.

On the left side of the Watch List page you can choose between custom Lists, Devices, Sources, or directly from folders of the Hierarchy. Choosing a new list will clear the chart and load the new points contained within that list into the table view. Each type of list is explained below:

  • Lists - A custom list of points that can be generated using the Watch List Builder. Lists can be generated that are dynamic and based on query parameters.
  • Devices - An arbitrary group assigned to data points often used to refer to a machine name. Every data point has a single arbitrary device name, which can be edited while editing the point.
  • Sources - A "place" from which data is received. Data points must be created on data sources. Every data point has exactly one data source.
  • Hierarchy - The point hierarchy can be used to arrange the points in your system in meaningful ways. Data points can be arranged in a hierarchical folder like structure regardless of their data source or device name.

All column names in the Watch List can be clicked to sort ascending or descending by the selected field. Point values will flash with the live value as they update in realtime.

If the viewed list is a custom Watch List, you can click the edit button to open the Watch List Builder page and edit the points on the selected Watch List.

When viewing Lists, the top option will allow you to create a New watch list and you will be linked to the Watch List Builder page.

You can click the checkbox on the far left of each row in the Watch List to add a data point to the Chart and Compare pane. By default a point will be added to the chart using the the left axis and the line color set in the point’s properties. This can be changed by checking the Configure next point checkbox beneath the chart. This allows you to set the color, chart type, and axis for the next point(s) added to the chart above. This can be useful if you want to chart a few points on the left axis that share a similar range and other points on the right axis that share a far different range.

Once added to the chart you can also click the Stats tab to view a comparison of statistics over the selected time range in the Header Bar. The time range in the header bar controls the time period displayed in the chart as well.



The chart icon in the row for a point opens a dialogue to give statistics, charts and values for that singular point over the time range.

If a data point is Settable a wrench icon will display on the row that will open a popup dialog you can use to manually set a data point’s value.

To the far right of the row is an icon to linked to the Data Point Details page for that data point.
