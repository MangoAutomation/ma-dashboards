maPointValues
directive in module ngMango

Description

<ma-point-values point="point1" values="point1Values" from="from" to="to" rollup="AVERAGE" rollup-interval="1 minutes"> </ma-point-values>

  • The <ma-point-values> directive returns an array of historical values on a data point to its values attribute.
  • <ma-point-values> is passed a to & from values from a date picker.
  • Additionally, rollup & rollup-interval can be used to average the data
  • You can use the point-xid property or pass in a point from <ma-point-list>.
  • View to/from Demo / View latest values Demo

Parameters

ParamTypeDetails
pointobject

Inputs a point object from <ma-point-list>

points
(optional)
object

Alternatively you can input an array of points from <ma-point-query>

point-xid
(optional)
string

Alternatively you can pass in the xid of a point to use.

valuesarray

Outputs the array of historical point values.

fromobject

The starting time to query the point values over a time period.

toobject

The ending time to query the point values over a time period.

latest
(optional)
number

Rather then to/from you can choose to use this property with the latest X number of values.

realtime
(optional)
boolean

Used with the latest attribute, if set to true the latest X number of values will update as new values are pushed to a data point.

rollup
(optional)
string

The statistical operation to apply to the values over the given rollup-interval. This will effect the outputted values. Rollup possibilities are:

  • POINT_DEFAULT
  • NONE
  • AVERAGE
  • DELTA
  • MINIMUM
  • MAXIMUM
  • ACCUMULATOR
  • SUM
  • FIRST
  • LAST
  • COUNT
  • INTEGRAL
rollup-interval
(optional)
string

The interval used with the rollup. Format the interval duration as a string starting with a number followed by one of these units:

  • years
  • months
  • weeks
  • days
  • hours
  • minutes
  • seconds
  • milliseconds
rendered
(optional)
boolean

If set to true the values will be outputted in the points text rendered value format.

converted
(optional)
boolean

If set to true the values will be outputted in the points converted format.

timezone
(optional)
boolean

The output date will have the given timezone.

use-cache
(optional)
boolean

Set to true to use cached values.

on-values-updated
(optional)
function

Pass in a function or expression to be evaluated when the values update. (eg. on-values-updated="$ctrl.valuesUpdated($values)")

date-format
(optional)
string

If you are passing in to/from as strings, then you must specify the moment.js format for parsing the values.

timeout
(optional)
number

If provided you can set the timeout (in milliseconds) on the querying of point values. If not supplied the Mango system default timeout will be used.

auto-rollup-interval
(optional)
boolean

If set to true the rollup interval will automatically be set based on the to-from duration and rollup type. DELTA rollup type will have a more chunked rollup interval. If turned on the manually set rollup-interval value will be ignored (defaults to false).
