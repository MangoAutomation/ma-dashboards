maPointList
service in module ngMango

Description

<ma-point-list limit="200" ng-model="myPoint"></ma-point-list>

  • Displays a list of data points which you can select from.
  • You can optionally supply the query attribute to filter the list of points using an object where the keys are the property names to filter on, or supply a RQL query string inside single quotes.
  • All data points added to your Mango system will display by default, unless you set the limit property.
  • View Demo

Usage

<ma-point-list limit="200" ng-model="myPoint2" query="{name: 'meta'}"></ma-point-list>

Parameters

ParamTypeDetails
ng-modelobject

Variable to hold the selected data point.

init-point
(optional)
boolean

Enables auto selecting of the first data point in the list. (Defaults to true)

query
(optional)
object

Filters the results by a property of the data points (eg: {name: 'meta'} returns data points containing the string 'meta' in the name property)

sort
(optional)
array

Sorts the resulting list by a properties of the data points. Passed as array of strings. (eg: ['-xid'] sorts descending by xid of the data points. Defaults to ['name'])

start
(optional)
number

Sets the starting index for the resulting list. Must be used in conjunction with a limit value. (Defaults to 0)

limit
(optional)
number

Limits the results in the list to a specified number of data points. Limit takes place after query and sorting (no limit by defualt)
