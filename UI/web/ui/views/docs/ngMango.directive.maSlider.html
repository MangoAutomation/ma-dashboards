maSlider
directive in module ngMango

Description

<ma-slider></ma-slider>

  • This component will display a slider that you can use to set a numeric data point's value.
  • You can update the value by dragging left and right (or up and down in vertical mode).

Usage

<ma-slider point="myPoint" vertical="true" min="0" max="25" step="5"></ma-slider>

Parameters

ParamTypeDetails
pointpoint

Select the point to use with the slider by passing in a point object.

point-xidstring

Select the point to use with the slider by passing in a point's xid as a string.

minnumber

Sets the min value for the slider (lower limit defaults to 0).

maxnumber

Sets the max value for the slider (upper limit defaults to 100).

stepnumeric

Sets the interval each tick in the slider will move the value by. (defaults to 1)

labelstring

Set the label to display with the slider. If left off the point name will be used. Set to none to hide the label.

verticalboolean

Enables vertical mode display of the slider.

invertboolean

Enables inverted mode and reverses the max / min.

discreteboolean

Enables discrete mode and the slider will display a value on the pin as you drag. In discrete mode the point's value will only be updated when you stop dragging.

hide-inputboolean

Disables displaying a numeric input next to the slider. (defaults to false)
