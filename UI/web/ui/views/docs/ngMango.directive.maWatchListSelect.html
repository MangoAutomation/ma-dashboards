maWatchListSelect
directive in module ngMango

Description

<ma-watch-list-select watch-list="myWatchlist"></ma-watch-list-select>

  • The <ma-watch-list-select> component can be used to load watch list data onto a custom page.
  • Can be combined with <ma-watch-list-chart> to display the watch list's custom chart designed on the watch list page.

Usage

<ma-watch-list-select no-select="true" watch-list-xid="{{watchlistXID}}" watch-list="myWatchlist"></ma-watch-list-select>

Parameters

ParamTypeDetails
ng-modelobject

Variable holds the resulting watch list object.

watch-list-xid
(optional)
string

Set to the XID of a watch list to auto load.

select-first
(optional)
boolean

Set to false to not auto select a the first watch list. (Defaults to true)

ng-disabled
(optional)
boolean

Set to true to disable the dropdown input. (Defaults to false)

query
(optional)
object

Filters the results by a property of the watch list object

sort
(optional)
array

Sorts the resulting list by a property of the watch list object. Passed as array of strings. (eg: ['-xid'] sorts descending by xid)

start
(optional)
number

Sets the starting index for the resulting list. Must be used in conjunction with a limit value. (Defaults to 0)

limit
(optional)
number

Limits the results in the list to a specified number of watch lists. Limit takes place after query and sorting (no limit by default)

parameters
(optional)
string

Pass in parameters from <ma-watch-list-paremeters>

on-points-change
(optional)
string

Pass in a function or expression to be evaluated when the points change. (eg. on-points-change="$ctrl.pointsChanged($points)")
