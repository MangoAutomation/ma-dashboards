maDuration
filter in module ngMangoFilters

Description

Converts an input in milliseconds to a momentJS duration object that defines a length of time.

  • MomentJS duration methods can be called using the filter syntax to call functions:
    • 67223455ms is {{67223455|maDuration:'humanize'}}
  • View Filters Demo

Usage

In HTML Template Binding

{{ maDuration_expression | maDuration:functionName:arg1:arg2:arg3 }}

In JavaScript

$filter('maDuration')(ms, functionName, arg1, arg2, arg3)

Parameters

ParamTypeDetails
