maMoment
filter in module ngMangoFilters

Description

Converts a timetamp to a momentJS object that can be formatted and manipulated.

  • Moment methods can be called using the filter syntax to call functions:
    • Three days from now is {{'now' | maMoment:'add':3:'days' | maMoment:'format':'LLL'}}
    • {{myPoint.time | maMoment:'format':'ll LTS'}}
  • View Filters Demo

Usage

In HTML Template Binding

{{ maMoment_expression | maMoment:functionName:arg1:arg2:arg3 }}

In JavaScript

$filter('maMoment')(date, functionName, arg1, arg2, arg3)
