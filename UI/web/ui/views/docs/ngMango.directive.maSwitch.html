maSwitch
directive in module ngMango

Description

<ma-switch></ma-switch>

  • This component will display a switch that can be used to toggle a binary data point's value.
  • It can be set to display either radio buttons, checkbox or toggle switch.

Usage

<ma-switch point="myPoint" display-type="switch"></ma-switch>

Parameters

ParamTypeDetails
pointpoint

Select the point to use with the switch by passing in a point object.

point-xidstring

Select the point to use with the switch by passing in a point's xid as a string.

display-typestring

Sets the display type of the binary switch. Options are radio, checkbox or switch.
