maWatchListChart
directive in module ngMango

Description

<ma-watch-list-chart></ma-watch-list-chart>

  • Use <ma-watch-list-chart> to display the watch list's custom chart on a custom page.
  • Use with <ma-watch-list-select> and pass in data from a watch list object.

Usage

<ma-watch-list-select no-select="true" watch-list-xid="WatchList323" watch-list="myWatchlist"></ma-watch-list-select>
<ma-watch-list-chart flex add-checked="myWatchlist.data.selectedPoints" chart-config="myWatchlist.data.chartConfig" to="dateBar.to" from="dateBar.from" rollup-type="dateBar.rollupType" rollup-interval-number="dateBar.rollupIntervals" rollup-interval-period="dateBar.rollupIntervalPeriod" chart-height="450px"></watch-list-chart>

Parameters

ParamTypeDetails
pointsarray

Array of points to add to the chart (myWatchlist.data.selectedPoints).

watch-listobject

Watch list object, passed in from watch-list-get

tostring

Timestamp to start charting. Can be from dateBar or <ma-date-range-picker>.

fromstring

Timestamp to end charting. Can be from dateBar or <ma-date-range-picker>.

rollup-typestring

Rollup type.

rollup-interval-numberstring

Rollup interval number.

rollup-interval-periodnumber

Rollup interval unit.

chart-heightstring

Height of the chart. Specify with px or % (400px).

edit-mode
(optional)
boolean

Set to true to display chart customization controls. (Defaults to false).

stats-tab
(optional)
boolean

Set to true to display stats tab. (Defaults to false_.

export
(optional)
boolean

Set to true to display chart export and annotation options. Defaults to false.

legend
(optional)
boolean

Set to false to hide the legend. (Defaults to true)

balloon
(optional)
boolean

Set to false to hide the balloon. (Defaults to true)

on-values-updated
(optional)
function

Pass in a function or expression to be evaluated when the values update. (eg. on-values-updated="$ctrl.valuesUpdated($values)")
