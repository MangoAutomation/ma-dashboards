maStatisticsTable
directive in module ngMango

Description

<ma-statistics-table statistics="statsObj"></ma-statistics-table>

  • <ma-statistics-table> will display a formatted data table with the values and timestamps from a statistics object.
  • View Demo

Usage

<ma-point-statistics point="myPoint" from="from" to="to" statistics="statsObj"></ma-point-statistics>
<ma-statistics-table statistics="statsObj"></ma-statistics-table>

Parameters

ParamTypeDetails
statisticsobject

Input the statistics object from <ma-point-statistics>

hide-starts-and-runtimes
(optional)
boolean

If set to false, starts and runtimes statistics will not display for a binary/multistate point (Defaults to true)

timezonestring

The display date will have the given timezone.
