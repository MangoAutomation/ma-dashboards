maNow
service in module ngMango

Description

<ma-now update-interval="1 SECONDS" output="time"></ma-now>

  • This directive will output the current browser time as a Moment.js date object.
  • View Demo

Usage

<ma-now update-interval="1 SECONDS" output="time"></ma-now>
<ma-clock style="width: 100%; height: 200px;" time="time1" text="Browser timezone">
</ma-clock>

Parameters

ParamTypeDetails
outputobject

Variable to hold the output date.

browserTimezoneobject

set to the browsers timezone (guessed)

update-intervalstring

The date will update on this given interval.

time-zonestring

The output date will have the given timezone. Format the interval duration as a string starting with a number followed by one of these units:

  • years
  • months
  • weeks
  • days
  • hours
  • minutes
  • seconds
  • milliseconds
