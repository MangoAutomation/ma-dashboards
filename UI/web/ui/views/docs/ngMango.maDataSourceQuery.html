maDataSourceQuery
service in module ngMango

Description

<ma-data-source-query data-sources="dataSourcesArray" query="{name: 'meter'}" limit="100"></ma-data-source-query>

  • Outputs an array of Mango data sources.
  • View Demo

Usage

<ma-data-source-query data-sources="dataSourcesArray" query="{name: 'meter'}" limit="100"></ma-data-source-query>

Parameters

ParamTypeDetails
data-sourcesarray

Variable to hold the array of outputted data source objects.

query
(optional)
object

Filters the results by a property of the data source object (eg: {name: 'meta'} returns data sources containing the string 'meta' in the name property)

sort
(optional)
array

Sorts the resulting list by a properties of the data source object. Passed as array of strings. (eg: ['-xid'] sorts descending by xid of data sources. Defaults to ['name'])

start
(optional)
number

Sets the starting index for the resulting list. Must be used in conjunction with a limit value. (Defaults to 0)

limit
(optional)
number

Limits the results in the list to a specified number of data sources (no limit by defualt)
