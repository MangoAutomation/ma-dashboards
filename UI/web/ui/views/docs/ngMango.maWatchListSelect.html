maWatchListSelect
service in module ngMango

Description

<ma-watch-list-select watch-list="myWatchlist"></ma-watch-list-select>

  • The <ma-watch-list-select> component can be used to load watch list data onto a custom page.
  • Can be combined with <ma-watch-list-chart> to display the watch list's custom chart designed on the watch list page.

Usage

<ma-watch-list-select no-select="true" watch-list-xid="{{watchlistXID}}" watch-list="myWatchlist"></ma-watch-list-select>

Parameters

ParamTypeDetails
watch-listobject

Variable holds the resulting watch list object.

watch-list-xid
(optional)
string

Set to the XID of a watch list to auto load or bind a string for dynamic switching (as shown above).

no-select
(optional)
boolean

Set to true to hide the dropdown select from the page, but still availble for loading watchlist data. (Defaults to false)

points
(optional)
array

Array of point objects contained in the watch list object.

select-first
(optional)
boolean

Set to false to not auto select a the first watch list. (Defaults to true)

disabled
(optional)
boolean

Set to true to disable the dropdown input. (Defaults to false)
