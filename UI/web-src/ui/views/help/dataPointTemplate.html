A Data Point Property Template allows a way to manage and assign templates to data points which override/set certain property values. Point property sections which are overridden by a Property Template are:

  • Point properties — Except for Units
  • Logging properties
  • Text renderer properties
  • Chart renderer properties

Support in Mango 3.6 and up

Support for property templates in Mango version 3.6 and up is deprecated. The new data point editor and CSV import/export feature negates the need for assigning property templates to points.

Saving a data point in the new data point editor will remove the template from the data point, all property values will be maintained.
