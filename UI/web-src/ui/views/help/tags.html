Data point tags allow you to add additional tags to the data point over and above the points device name and name. This gives you more flexabilty to create complex data structures. Tags use a {key: value} structure eg: Site Name: Building 1. A data point can have many different tags but only one value per tag key. Tags can be used to make dynamic watchlists & pages and provide an intuative way to query and filter data.
