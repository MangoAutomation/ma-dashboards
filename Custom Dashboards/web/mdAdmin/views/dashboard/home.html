Mango Custom Dashboards

Welcome to Mango's Custom Dashboards Module

Custom Dashboards are a powerful way to create dynamic visualization screens for your customers. They are designed to both impress and offer superior functionality and performance. The development environment is designed for rapid development and deployment.

Start off by exploring the live demos under the Examples menu item. You can then copy mark-up into the play area or copy the code into an entirely new page. The Custom Dashboards Module now supports user created pages and total customization of the menu. Learn more on how to create pages and edit the menu.

The new custom dashboards module leverages Angular, the powerful javascript framework, along with Angular Material to provide a set of UI components and a theming model for creating beautiful and modern frontends that utilize the mango backend. In reality you will not have to use any javascript at all. You will only need to understand how to use our custom HTML components.

We have added documentation for every component used to create a dashboard. Detailed descriptions of every parameter and attribute is available in the API Docs section.

For tips on where to find and edit the files see our online help site. To get help or ask questions see our forum.

Have fun!
