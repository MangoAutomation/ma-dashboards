Events
service in module maServices

Description

Provides a service for retrieving, adding, and acknowledging events/alarms

  • Used by <ma-events-table>

Methods

  • acknowledge(id)

    Acknowledges a single event by passing in an event id.

    Parameters
    ParamTypeDetails
    idobject

    Passed in an object with an id property cooresponding to the event id.

    Returns
    object

    Returns an event object with an id property if successful.

  • acknowledgeViaRql(rql)

    Acknowledge many events using an RQL string as a parameter

    Parameters
    ParamTypeDetails
    rqlobject

    Object with an rql string property equal to the rql used to find all the events that you wish to acknowledge.

    Returns
    object

    Returns an object with a count property equal to the number of events acknowledged.
