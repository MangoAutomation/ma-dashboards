maPieChart
service in module maDashboards

Description

<ma-pie-chart style="height: 300px; width: 600px" values="values"></ma-pie-chart>

  • This directive will display a pie chart for visualizing ratios.
  • Values are provided as an array of objects.
  • Note, you will need to set a width and height on the element.
  • View Demo

Usage

<ma-pie-chart style="height: 300px; width: 600px" values="[ { "value": 30, "text": "hot", "color": "#ff9300" }, { "value": 70, "text": "cold", "color": "#942192" } ]"
options="{depth3D:15,angle:30}"></ma-pie-chart>

Parameters

ParamTypeDetails
valuesarray

Takes in an array of value objects to use in the pie chart. Each value object can contain the following properties:

  • value - Numeric Value
  • text - Text label
  • color - Color of the pie piece
options
(optional)
object

extend AmCharts configuration object for customizing design of the chart (see amCharts)

value-labels
(optional)
object

For use with the multi state data point render map. (see multi-state pie chart demo)
