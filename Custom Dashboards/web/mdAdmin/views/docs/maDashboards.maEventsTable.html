maEventsTable
service in module maDashboards

Description

<ma-events-table></ma-events-table>

  • Displays a list of Events in a table format.

Usage

<ma-events-table event-type="eventType" limit="25"></ma-events-table>

Parameters

ParamTypeDetails
event-typestring

Query via eventType

point-idstring

Query via pointId, not to be used with other query by attributes

alarm-levelstring

Query via alarmLevel

limitnumber

Set the initial limit of the pagination
