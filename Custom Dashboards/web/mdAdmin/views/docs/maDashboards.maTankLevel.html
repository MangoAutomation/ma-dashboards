maTankLevel
service in module maDashboards

Description

<ma-tank-level point="myPoint" color="" max="" style="width:300px; height:200px;"></ma-tank-level>

  • This directive will display a custom data visualization which represents a tank.
  • The volume of 'liquid' within the tank increases in height with increases in point value.
  • Not you must set a width and height on the element.
  • View Demo

Usage

<ma-tank-level point="myPoint" color="{{choosenColor}}" max="{{max}}" style="width:300px; height:200px; position:absolute; top:0; left:0px; z-index:2">
</ma-tank-level>

Parameters

ParamTypeDetails
pointobject

The point object with the live value provided by <ma-get-point-value>.

max
(optional)
number

Sets the max value for a 100% filled tank.

color
(optional)
string

Sets the fill color for the tank visualization.

options
(optional)
object

Extend amCharts configuration object for customizing the design of the tank.

value
(optional)
number

Allows you to set the tank's fill height to a value that is not provided by the point attribute. Only use without the point attribute.
