This example showcases using angular material radio button components to set the value of multistate data points. Set the ng-model to the points valueFn property and set ng-model-options="{getterSetter:true}".

The first example shows how to loop through the text renderer map of the multistate point to get the various options. The second example shows how to use fixed values of 1-5.
