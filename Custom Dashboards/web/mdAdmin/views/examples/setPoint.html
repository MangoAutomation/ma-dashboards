The <ma-set-point-value> directive will create a form input element to set the value of a point. It can handle numeric, binary, and multistate point types and will create an appropriate interface element for each point type.

Alternatively you can set the value of a point by calling the setValue method on a point. This function can be called from withing a ng-click expression for example.
