There is a watchdog timer which resets every time an API response or WebSocket frame is received. Check the value of mangoWatchdog.loggedIn to see if the watchdog timer has timed out.

If your page template is setup to display a message when the watchdog times out then you may want to call mangoWatchdog.disable() on pages which do not make repeated requests.

See API Documentation for the mangoWatchdog service.
