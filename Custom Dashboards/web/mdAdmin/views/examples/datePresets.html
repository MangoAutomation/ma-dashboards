Use the <ma-date-range-picker> directive to insert a date range preset picker. This enables you to choose from a list of commonly used date ranges, such as "Today so far" or "Previous week". Set the update-interval attribute to have it update automatically.

Use the <ma-date-picker> directive to display a date/time picker, note that you can also add it as an attribute to an existing <input> tag.

You can tie the date range picker and the date pickers together using the from and to attributes, and the ng-model attributes
