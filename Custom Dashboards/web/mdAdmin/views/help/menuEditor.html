Custom pages can be created from within the app's CMS environment, using our built in Code Editor and UI Component Library to create and link new pages to the app. Mango's HTML Web Components combined with Angular templating logic allows you to create simple or complex dynamic pages that pull data from Mango's database and visualize the data as responsive interface elements, charts and tables. Interactive Demos and Examples can be found under the Examples menu section. Full documentation of all parameters of each component can be found under API Docs.


The Page Editor

The Page Editor can be found under Settings > Edit Pages in the menu. You can use this tool to manage all of your custom pages.

The inputs above the button row allow you to choose a pre-existing page to edit, update the name of a page, and set the page's permissions. Read / Edit Permissions can be used to prevent access to a page.

The first button will Save your changes to the page you are currently editing. The Revert button will reset the code editor to the last saved state. The Delete button will remove a page from the sytem after confirming. The New Page button will set-up a blank page for you to work with. Be sure to Save changes to the previous page first.

Before you can Save a new page you will need to give it a Name.

You can Toggle on and off the Editor Pane and Preview Pane allowing you to view just one pane or both in a side-by-side view.



Permissions

By default, all pages that are created can be viewed by all users and require the user to have the edit-pages permission to edit them. If you want to restrict the pages to be viewable only by admin users, change the read permission to superadmin.

Users with the edit-menus permission can edit the menus. When editing the menu items you can set a permission for each menu item, if the user does not have this permission the menu item will not be shown.


Adding a Menu Item

The Add Edit Menu Item button will open up the Edit Menu Item dialog. From here you will give the menu item a Parent Item, URL, Label, and Menu Icon. The entire menu is completely customizable from the Menu Editor under Settings > Edit Menu in the menu. You can Hide/Show menu items, Edit menu item Labels and Icons, Reorder menu items, add New menu items and Delete custom menu items. For more information see the Menu Help Page.
