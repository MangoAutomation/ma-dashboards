The Data Point Details page allows you drill down into the life a single Data Point. A Data Point can be selected using the Point Selector in the upper left pane. You can filter the dropdown list by typing either the Data Point Name or Device Name (caps sensitive). The results will update as you type. The Live Value, XID, Timestamp of Last Update, Point Hierarchy Folder Path and Data Source of the currently selected point will be displayed in this pane. You can also manually set a point’s value or link to the data point edit page using the EDIT button.

The Point Value History pane will show you the point’s values either in Realtime Mode or over the Selected Time Range in the Header Bar. By default Realtime Mode is enabled which will show the last 100 values logged to a data point’s history.

If you turn this mode off by clicking the Realtime Toggle Button you will see the point values using the time range from the header and Rollup Settings from the Date Bar.

The Point Statistics pane will show you statistics over the Selected Time Range. It will show different statistics depending on the type of data point selected.

The Point Comments pane will show any User Notes specific to the selected data point. Click the Add Button to enter a new note for the data point and it will be shown on the table.



The Chart will show a visualization of the data points values over the selected time range. The Rollup Settings from the Date Bar will be used to average the data if necessary. You can choose between different Chart Types and line smoothing options using the drop down selector in the upper right corner. Additionally, you can export an image or data file by clicking the Download Icon hovering over the upper right corner of the chart.

Lastly, at the bottom of the page you will see any Events specific to the selected data point. See the Events Help page for more information on the Events Table.
