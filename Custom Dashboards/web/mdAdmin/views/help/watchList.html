The Watch List page allows you to compare many data points at the same time in a table view. Powerful query and parameter based watchlists can be built using the Watchlist Builder, and later viewed on the this page.

On the left side of the Watch List page you can choose between custom Watch Lists, Devices Names, Data Sources, or directly from folders of the Point Hierarchy. Choosing a new list will clear the chart and load the new points contained within that list into the table view. Each type of list is explained below:

  • Watch Lists - A custom list of points that can be generated using the Watchlist Builder. Lists can be generated that are dynamic and based on query parameters.
  • Device Names - An arbitrary group assigned to data points. Typically used to refer to a machine name. Data sources can have multiple device names, grouping points within a single data source.
  • Data Sources - A "place" from which data is received. Virtually anything can be a Data Source, insofar as the communications protocol is supported by the application. Examples include Modbus Data Sources, SNMP Data Sources, and HTTP Data Sources.
  • Point Hierarchy - The point hierarchy can be used to arrange the points in your system in meaningful ways. Data points can be arranged in a hierarchical folder like structure regardless of there data source or device name.

All column names on the Watch List Table can be clicked to sort ascending or descending by the selected field. Point values will flash with the live value as they update in realtime.

If you have a Custom Watch List selected you can click the edit button to open the Watch List Builder page and edit the points on the selected Watch List.

Under Lists in the Watchlist Chooser the top option will allow you to create a New Watchlist and you will be linked to the Watchlist Builder page.

You can click the checkbox on the far left of each row on the Watch List Table to add a data point to the Chart and Compare pane.

Once added to the chart you can also click the Stats tab to view a comparison of statistics over the Selected Time Range in the Header Bar.



By default a point will be added to the chart using the the left axis and the line color set in the point’s properties. This can be changed by setting the Next Axis radio button to something other than left. The next point added to the chart will use the newly selected axis. This can be useful if you want to chart a few points on the left axis that share a similar range and a some other points on the right axis that share a far different range.

If you check the Pick Colors checkbox you can set the line color used by the next data point added to the chart. You can also change axis colors using the other color inputs. These settings will be stored next time you return to the watch list.

The Selected Time Range will be used on the Chart, Stats Tab, and Stats/History popup which can be accessed by clicking the Graph Icon to the right of the checkbox. Clicking this icon will open a popup dialog showing statistics, a line chart and a point value history for the data point, corresponding to the Selected Time Range in the Header Bar.



If a data point is Settable a Wrench icon will display on the Watchlist Table Row that will open a popup dialog you can use to manually set a data point’s value. The appropriate interface element will be shown for Numeric vs. Binary/Multistate/Alphanumeric type data points.



To the far right of the Watchlist Table Row you can click the icon to link to the Data Point Details page for that data point.
