Mango Automation is a powerful tool for a wide variety of applications. If you're interested in integrating, evaluating, and visualizing multiple data streams over time, you'll be amazed at what you can accomplish after only a little learning.

Our UI has been completely rebuilt to offer a modern single page application experience. Many new features have been introduced including a fully responsive component based UI framework, a new advanced watchlist building and viewing system, and the ability to create custom dashboard pages and visualizations all within the app.

Navigation

The menu on the left is used to access all pages. Our new pages as well as the legacy UI pages can be accessed via the sidebar menu. The username and email will be displayed at the top of the menu, under the logo and server time. You can click the usernamedrop down to logout of Mango. The sidebar menu will automatically hide on mobile views. If you like you can click the three dashes menu icon to hide/expand the menu manually when viewing on smaller desktop screens.

Pages to change theme colors, switch the logo as well as edit the menu structure itself can be found under the Settings menu item. The Watch List Builder and Page Editor can also be found under Settings. Lastly, all legacy pages can be viewed from menulinks under Settings.
